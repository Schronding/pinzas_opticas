# Simulador de Pinzas Ópticas

  

Este proyecto simula el comportamiento de una partícula en una trampa óptica y compara los resultados con datos experimentales.

  

## Instalación (Cómo clonar este proyecto)

  

### 1. Clonar el repositorio:

```bash

git clone [https://github.com/TU_USUARIO/pinzas_opticas.git](https://github.com/TU_USUARIO/pinzas_opticas.git)

cd pinzas_opticas

```

  

### 2. Instalar dependencias del sistema:

Revisa el archivo `system_requirements.txt`.

Si estás en Linux, asegúrate de tener Tkinter: `sudo apt-get install python3-tk`

### 3. Configurar entorno Python:

```
# Crear entorno virtual
python3 -m venv venv

# Activar entorno (Linux/Mac)
source venv/bin/activate

# Activar entorno (Windows)
venv\Scripts\activate

# Instalar librerías
pip install -r requirements.txt
```
